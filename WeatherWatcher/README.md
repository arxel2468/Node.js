# Weather Watcher❄️

A clean, single-screen dashboard that dynamically fetches live data from a public API (such as OpenWeather), processes the JSON payload, and updates a beautiful UI based on the time of day.

## Project Structure
```
my-awesome-project/
├── .gitignore
├── README.md
├── index.html
├── assets/
│   └── images/
├── src/
    ├── css/
    │   └── main.css
    └── js/
        ├── app.js
        ├── config.js
        ├── utils.js
        └── services.js
```

## Features
- Current weather conditions
- Temperature, humidity, wind speed
- Weather icons for visual representation

## How to Use
1. Obtain an API key from [OpenWeather](https://openweathermap.org/api).
2. Replace `YOUR_API_KEY` in the code with your actual API key.
3. Run the app to see weather updates for your location.
